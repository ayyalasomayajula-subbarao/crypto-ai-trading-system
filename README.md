# Crypto AI Trading System

Decision-first crypto trading engine focused on:
- WIN vs LOSS probability
- Market regime detection
- Risk gating
- Capital preservation

## Features
- Multi-timeframe analysis (1H / 4H / 1D / 1W)
- Decision model (TP vs SL outcome)
- Regime-aware filtering (ADX)
- Risk & guardrails engine
- Live UI dashboard

## Note
Training data and models are excluded from the repository.
